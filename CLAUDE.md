# Правила проекта

## Контекст

TODO-приложение для личного использования.
Стек: HTML, CSS, JavaScript. Без фреймворков и библиотек.
Уровень разработчика: начинающий.

## Структура

- index.html — разметка
- style.css — стили
- app.js — вся логика (addTask, deleteTask, toggleTask, renderTasks, filterTasks)
- storage.js — работа с localStorage

## Правила

- Объясняй изменения простыми словами
- Не добавляй библиотеки и фреймворки
- Не усложняй без необходимости
- Не меняй файлы, которые не относятся к задаче
- Если нужно изменить что-то за пределами задачи — сначала скажи что и зачем
- Код должен быть понятен новичку
